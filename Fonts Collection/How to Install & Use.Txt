
== HOW TO INSTALL ==

Simply copy all *.ttf files from the selected folder
& past them in "C:\Windows\Fonts\" folder.

== HOW TO USE ==

To use these fonts simply select you desired font from
fonts dialog box shown in any application. In VB6, select
font from Control's properties window.
